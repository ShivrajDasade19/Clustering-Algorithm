Abstract

This project applies clustering algorithms such as K-Means, Hierarchical Clustering, and DBSCAN to analyze student performance data. The dataset is grouped into three clusters: Excellent Students, Average Students, and At-Risk Students. The objective is to discover hidden patterns and classify students based on academic performance without predefined labels.

Introduction

Clustering is an unsupervised machine learning technique used to group similar data points together. In this project, student performance data is analyzed using clustering algorithms to identify patterns in academic behavior. The main goal is to segment students based on grades, study time, absences, and academic records.

Objectives

  • Group similar students into clusters.
  • Discover hidden academic performance patterns.
  • Maximize intra-cluster similarity.
  • Minimize inter-cluster similarity.
  • Reduce data complexity.
  • Support data-driven decision making.

Dataset Description

The dataset used is the Student Performance Dataset obtained from the UCI Machine Learning Repository. It contains academic, demographic, and social attributes of students from Portuguese schools.

Important Features Used:

  • Study Time
  • Number of Failures
  • Absences
  • First Period Grade (G1)
  • Second Period Grade (G2)
  • Final Grade (G3)

Methodology

1. K-Means Clustering

K-Means partitions data into K clusters by minimizing the distance between data points and cluster centroids.

2. Hierarchical Clustering

Hierarchical clustering builds clusters in a tree-like structure using a dendrogram to visualize cluster formation.

3. DBSCAN

DBSCAN groups data points based on density and identifies noise or outliers effectively.

Results

The dataset was successfully divided into three clusters:

  • Cluster 1: Excellent Students
  • Cluster 2: Average Students
  • Cluster 3: At-Risk Students

These clusters help educators identify performance levels and provide targeted academic support.

Conclusion

This project demonstrates how clustering algorithms can effectively classify student performance data into meaningful groups. The results provide valuable insights that can assist educational institutions in improving academic strategies and student support systems.
